# Smart Biomedical Waste Segregation System

## 📖 Project Overview
This project presents a Smart Biomedical Waste Segregation and Monitoring System developed using Arduino Uno. The system automatically identifies and segregates waste into wet, dry, and metal categories using moisture and metal detection sensors. Bluetooth authentication provides secure access for authorized users.

## ✨ Features
- Automatic waste segregation
- Moisture detection
- Metal detection
- Servo motor controlled bin
- Bluetooth authentication
- Low-cost and easy to implement

## 🛠 Hardware Components
- Arduino Uno
- Moisture Sensor
- Metal Detector Sensor
- IR Sensor
- Servo Motor
- Stepper Motor
- Stepper Motor Driver
- Bluetooth Module
- Buzzer
- Power Supply

## ⚙️ Software Used
- Arduino IDE

## 🔄 Working Principle
1. User authentication through Bluetooth.
2. Waste is placed inside the system.
3. Sensors detect the waste type.
4. Arduino processes the sensor data.
5. Motor rotates the bin to the correct compartment.
6. Waste is collected automatically.
